# ⚓ JSONPort

**Explore • Validate • Transform JSON**

JSONPort is a lightweight web tool for exploring, validating, formatting, and searching JSON data.
It runs entirely in the browser and is designed to help developers inspect and work with JSON quickly and efficiently.

---

## 🚀 Live Demo

Try it here:

https://jsonport.in

---

## ✨ Features

* **Monaco JSON Editor**

  * Powerful code editor used in VS Code
  * Syntax highlighting
  * Line numbers and formatting support

* **JSON Validation**

  * Detects invalid JSON
  * Highlights the exact error location in the editor
  * Displays validation messages clearly

* **JSON Search with Path Detection**

  * Search keys or values
  * Shows the full JSON path
    Example:

    ```
    users[0].email
    users[1].id
    ```

* **Copy JSON Path**

  * One-click copy of any JSON path from search results

* **JSON Statistics**
  Displays useful metadata about the JSON structure:

  * File size
  * Number of lines
  * Object count
  * Array count
  * Maximum nesting depth

* **Format JSON**

  * Beautifies JSON with proper indentation

* **Minify JSON**

  * Compress JSON into a single line

* **Upload JSON Files**

  * Load JSON files directly from your system

* **Download JSON**

  * Export edited JSON easily

* **Clipboard Support**

  * Copy JSON content instantly

## 📦 Installation

Clone the repository:

```
git clone https://github.com/your-username/jsonport.git
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Open:

```
http://localhost:5173
```

---

## 🏗 Build for Production

Create a production build:

```
npm run build
```

Preview the production build:

```
npm run preview
```

## 🔒 Privacy

JSONPort processes all JSON **locally in the browser**.

* No data is sent to any server
* No JSON content is stored
